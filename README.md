# Sales-Dashboard-Excel
Excel Sales Dashboard with city-wise filtering, sales performance analysis, target achievement metrics, and automated reporting using pivot tables and macros.

This project is an interactive Excel dashboard built to analyze sales performance across multiple cities and sales executives. It helps track total sales, target achievement, and performance gaps using dynamic filters, charts, and pivot tables.

## Key Features
- City-wise filtering (Chennai, Delhi, Mumbai, Nagpur, Patna, Pune, Ranchi, Surat)
- Sales Executive-wise performance analysis
- Total Sales by Executive
- Target Hit % tracking
- Away From Target % analysis
- Interactive dashboards using buttons and slicers
- Automated calculations using Excel macros
- Pivot tables and dynamic charts for reporting

## KPIs & Metrics
- Total Sales  
- Target Hit Percentage  
- Away From Target Percentage  
- Sales Performance Comparison  

## Tools & Technologies
- Microsoft Excel (Macro-enabled .xlsm)
- Pivot Tables & Pivot Charts  
- Excel Formulas  
- VBA Macros (for automation & interactivity)  
- Data Cleaning & Data Preparation  

## Business Use Case
This dashboard can be used by sales managers and business analysts to:
- Monitor sales performance across cities  
- Identify top and underperforming sales executives  
- Track target achievement  
- Support data-driven business decisions  

## How to Use
1. Download the `.xlsm` file  
2. Open in Microsoft Excel  
3. Enable macros when prompted  
4. Use city buttons and dashboard tabs to interact with the report
